# ekko

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

---

## Local 세팅
API 호출 시 필요한 JWT 를 .env.local 에 세팅한다.
```
VUE_APP_JWT=eyJhbGciOiJIUzUxM...생략
```
